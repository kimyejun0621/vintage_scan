## 🔄 개선 사항

### 1. 브랜드별 구체적인 가격 범위 추가

**이전**: 가격 가이드라인 없음 → AI가 임의로 추정
**개선**: 브랜드/연도별 실제 시장 가격 범위 제시

#### Levi's 501 가격 가이드
```
1960s-70s Big E: $200-500 (₩260,000-670,000)
1980s 501XX: $120-250 (₩160,000-330,000)
1990s Valencia: $80-180 (₩105,000-240,000)
2000s-현재: $30-70 (₩40,000-95,000)
```

#### Supreme 가격 가이드
```
1994-2000 초기: $300-1000 (₩400,000-1,330,000)
2001-2005 중기: $150-400 (₩200,000-530,000)
2006-2010: $80-200 (₩105,000-265,000)
2021-현재: $30-100 (₩40,000-130,000)
```

#### Stussy 가격 가이드
```
1990s 초기: $100-300 (₩130,000-400,000)
1990s 중후반: $60-150 (₩80,000-200,000)
2000s: $40-100 (₩50,000-130,000)
2010s-현재: $25-60 (₩30,000-80,000)
```

### 2. 가격 조정 팩터 시스템 도입

**컨디션 팩터**
- Deadstock/NWT: +50%
- Excellent: +20%
- Good: 기준가
- Fair: -30%
- Poor: -60%

**희귀도 팩터**
- 레어 컬러/사이즈: +30-50%
- 리미티드 에디션: +40-100%
- 콜라보: +50-200%
- 일반: 기준가

**수요 팩터**
- 트렌디: +20-30%
- 인플루언서 착용: +30-50%
- 일반: 기준가
- 비인기: -20%

### 3. Few-Shot Learning 예시 추가

AI가 가격 계산 과정을 학습할 수 있도록 4가지 예시 추가:

**예시 1**: Levi's 1990s Good 컨디션
```
기본가 $120 → 조정 없음 → 최종 $120 (₩160,000)
```

**예시 2**: Supreme 2000 Excellent + 레어
```
기본가 $300 → +20% (+30%) → 최종 $468 (₩620,000)
```

**예시 3**: Levi's 2023 현행 Deadstock
```
기본가 $50 → +50% → 최종 $75 (₩100,000)
```

**예시 4**: Stussy 1995 Fair 컨디션
```
기본가 $150 → -30% → 최종 $105 (₩140,000)
```

### 4. 브랜드별 특화 정보 추가

각 브랜드의 진품 감별 포인트를 동적으로 추가:

#### Levi's 특화 가이드
- Big E vs Small e 구분
- Valencia/Turkish 공장 정보
- 케어라벨 변천사
- 501XX vs 501 차이
- LVC 복각판 주의사항

#### Supreme 특화 가이드
- 박스로고 폰트 변천
- 메이드인 태그 (USA → Canada → China)
- 워터마크 (2010s 이후)
- 시즌/콜라보 정보

#### Stussy 특화 가이드
- 로고 변천 (올드 스툴 → 현대)
- 빈티지 택 (노란색 → 흰색)
- 제작 방식 (스크린 → 디지털)
- 일본 라인 특징

### 5. 4단계 분석 체크포인트 구조화

**이전**: 간단한 체크리스트
**개선**: 단계별 상세 가이드

🔍 **1단계: 진위 확인**
- 스티칭 균일도
- 폰트 정확도
- 소재 품질
- 태그 위치
- 시대착오 체크

🔍 **2단계: 연식 추정**
- 케어라벨 형태 (시대별 변천)
- 생산 공장 정보
- 태그 스타일
- 로고 변천
- 메이드인 표기

🔍 **3단계: 컨디션 평가**
- 색 바램
- 얼룩/손상
- 프린트 상태
- 원단 상태
- 디테일 보존

🔍 **4단계: 희귀도 판단**
- 레어 컬러/사이즈
- 한정판 표기
- 콜라보레이션
- 특별 라인
- 생산 수량

### 6. Reason 작성 가이드 구체화

**이전**: "판단 근거를 한글 3-4문장으로"
**개선**: 구조화된 템플릿 제공

```
1) 진위/연식 판단 근거 (구체적 특징)
2) 컨디션 평가 (등급 명시)
3) 가격 근거 (계산 과정 설명)
```

**예시**:
"케어라벨의 은색 띠와 Valencia 공장 표기로 1990년대 제품으로 추정됩니다. 전체적으로 색 바램이 있으나 큰 손상은 없어 Good 컨디션입니다. 1990s 501 기본가($120)에 일반 컨디션을 적용하여 ₩160,000로 추정합니다."
